This is a sample post so we can see the blog working end to end. The publish date above is set
in the file's front matter (and its filename), which is how every post controls when it appears —
so migrated posts can keep their original dates.

A few things this layout already does:

- **Headings** render cleanly.
- **Lists** like this one.
- Links — for example, a [contact link](/contact/) to turn readers into leads.

> Blockquotes work too, in case you want to pull a line out.

When we migrate the real posts, each one becomes a Markdown file just like this in the `_posts`
folder, and it shows up automatically on the [blog index](/blog/).
